# ML-shiftintensive26
Модель для оценки предварительной цены автомобиля на рынке при продаже.   
Проект ШИФТ Интенсив, трек Машинное обучение (Classic).

## Установка (Linux/MacOS):

### 1. Инициализация переменного окружения

```
python3 -m venv .env
source .env/bin/activate
```
### 2. Установка зависимостей

```
pip install -r requirements.txt
```

### 3. Запуск

```
streamlit run inference/inference.py
```

После запуска в командной строке отобразится адрес, по которому нужно перейти в браузере для получения доступа к интерфейсу модели
